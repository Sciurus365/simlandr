# A simple simulation function for testing

A simple simulation function for testing

## Usage

``` r
sim_fun_test(arg1, arg2, length = 1000)
```

## Arguments

- arg1, arg2:

  Two parameters. `arg1` contains `ele1`; `arg2` contains `ele2` and
  `ele3`.

- length:

  The length of simulation.

## Value

A matrix of simulation results.
