# Modify a single simulation

Modify a single simulation

## Usage

``` r
modified_simulation(sim_fun, ele_list, default_list, bigmemory = TRUE, ...)
```

## Arguments

- ele_list:

  An `ele_list` object generated by
  [`fill_in_struct()`](https://sciurus365.github.io/simlandr/reference/fill_in_struct.md).
