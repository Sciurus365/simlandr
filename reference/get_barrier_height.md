# Summarize the barrier height from a `barrier` object

**\[deprecated\]**

This function was deprecated. Use
[`summary()`](https://rdrr.io/r/base/summary.html) instead. (See
[`summary.barrier()`](https://sciurus365.github.io/simlandr/reference/summary.barrier.md).)

## Usage

``` r
get_barrier_height(object, ...)
```

## Arguments

- object:

  A `barrier` object.

- ...:

  Not in use.

## Value

A vector (for a single barrier calculation result) or a `data.frame`
(for batch barrier calculation results) that contains the barrier
heights on the landscape.
