# Make a tidy `data.frame` from smooth 2d distribution matrix

Make a tidy `data.frame` from smooth 2d distribution matrix

## Usage

``` r
make_2d_tidy_dist(dist_2d, value = NULL, var_name = NULL)
```

## Arguments

- dist_2d:

  `kde2d` distribution.

- value:

  The value of the variable of interest.

- var_name:

  The name of the variable.

## Value

A tidy `data.frame`.
