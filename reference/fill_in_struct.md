# Fill a vector of values into a structure list.

Fill a vector of values into a structure list.

## Usage

``` r
fill_in_struct(vec, struct)
```

## Arguments

- vec:

  A vector of values.

- struct:

  A list with a certain structure.

## Value

A `ele_list` object.
