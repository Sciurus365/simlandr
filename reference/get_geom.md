# Get a ggplot2 layer from a barrier object

**\[deprecated\]**

This function was deprecated. Use
[`autolayer()`](https://sciurus365.github.io/simlandr/reference/autolayer.md)
instead. (See
[`autolayer.barrier()`](https://sciurus365.github.io/simlandr/reference/autolayer.barrier.md).)

## Usage

``` r
get_geom(object, ...)
```

## Arguments

- object:

  A `barrier` object.

- ...:

  Not in use.

## Value

A `ggplot2` layer that can be added to an existing landscape plot.
